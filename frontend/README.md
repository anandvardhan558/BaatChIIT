# BaatChIIT Frontend

Frontend for BaatChIIT, a real-time video conferencing platform built using React, WebRTC, and Socket.IO.

The application provides a modern meeting experience with video calling, screen sharing, real-time chat, participant management, and secure authentication.

## Live Demo

https://baat-chiit.vercel.app

---

## Features

* Secure Login & Signup
* JWT-based Authentication
* Real-Time Video Calling
* Screen Sharing
* Meeting Chat
* Participant Management
* Meeting History
* Responsive Design
* Modern Meeting Interface
* Real-Time Status Indicators

---

## Tech Stack

* React.js
* React Router
* Material UI (MUI)
* Axios
* Socket.IO Client
* WebRTC

---

## Getting Started

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the root of the frontend directory:

```env
REACT_APP_API_URL=http://localhost:8000
```

### Run Development Server

```bash
npm start
```

Application will be available at:

```text
http://localhost:3000
```

---

## Build for Production

```bash
npm run build
```
